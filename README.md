# TradingView-Binance-Pairs
Binance Pairs for TradingView ( USDT , BNB, ETH, BTC ) markets

simply run the script
**python3 main.py**

choose between markets 
* 1 for **USDT** 
* 2 For **BNB** 
* 3 for **ETH** 
* 4 for **BTC**
* **Any other number will Fetch all markets**
